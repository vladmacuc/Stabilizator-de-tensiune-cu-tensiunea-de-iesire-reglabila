# Stabilizator de Tensiune cu Tensiunea de Ieșire Reglabilă ⚡

Acest depozit conține documentația și fișierele de proiectare asistată de calculator (CAD) pentru un **Stabilizator de tensiune cu tensiunea de ieșire reglabilă**. Proiectul a fost realizat pentru a ilustra designul și funcționarea unui circuit stabilizator de tensiune.

## 📁 Conținutul Repository-ului

- **`CAD - Data Sheet/`** - Acest director conține fișele tehnice (datasheets) pentru componentele electronice utilizate în proiect.
- **`Proiect CAD final/`** - Aici se regăsesc fișierele sursă ale proiectului CAD (ex. scheme electronice, design PCB etc.).
- **`MacucVlad_CAD.docx`** - Documentația completă a proiectului în format editabil Word.
- **`MacucVlad_CAD.pdf`** - Documentația completă a proiectului în format PDF, gata pentru a fi vizualizată sau printată.

## 📖 Descriere

Proiectul prezintă etapele de proiectare și simulare pentru un stabilizator de tensiune care permite reglarea tensiunii de ieșire. Documentația inclusă (`.pdf` și `.docx`) explică în detaliu:
- Schema bloc și schema electronică a circuitului.
- Calculul și dimensionarea componentelor.
- Design-ul cablajului imprimat (PCB).
- Explicații referitoare la funcționarea circuitului și parametrii de ieșire.

## 🛠️ Cum să vizualizezi proiectul

1. Pentru a citi documentația teoretică și a vedea explicațiile, deschide fișierul **`MacucVlad_CAD.pdf`**.
2. Pentru a vedea specificațiile componentelor, navighează în folderul **`CAD - Data Sheet`**.
3. Pentru fișierele de design, descarcă folderul **`Proiect CAD final`** și folosește mediul de dezvoltare/proiectare CAD specific în care a fost lucrat (de ex. KiCad, Altium, Proteus, Eagle etc.).
